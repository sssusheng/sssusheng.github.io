python数据结构之链表(Linked List)

发表于 2019-10-21 更新于 2020-03-02

​ 链表(Lindked List)是一种链式数据结构。与数组相比，它并不需要一块连续的内存空间，它通过指针将一组零散的内存块串联起来使用，是可以无限扩容的。

链表(Linked List)

链表的概念

常用的链表有三种：单向链表，双向链表和循环双向链表。

单向链表的节点中只存储了数据和指向下一个节点的指针；

双向链表中还多存储了前一个节点的指针；

循环双向链表则是在双向链表的基础上，头节点的前指针指向尾节点，尾节点的后指针指向头节点，形成一个封闭的链表结构。

链表的操作

操作 单向链表 双向链表
向链表尾部新增元素 O(1) O(1)
向链表头部新增元素 O(1) O(1)
按值删除元素 O(n) O(n)
按节点删除元素 O(n) O(1)
